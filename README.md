# REST api for ChattyChat 
  * Git: https://github.com/ollilappalainen/chattychat

### Technologies
  * JavaScript
  * Node.js
  * Express
  * Mongoose
  * MongoDB

### Instructions
  * Install local MongoDB server
  * Name your db as 'chattychatdb'
  * Run on terminal 'yarn install'
  * Run on terminal 'yarn start'
